Quick Guide

üì∑ Camera Settings

Camera IP/Host: Your camera's network address

Channel: Usually 0 for single cameras, 0-7 for NVR systems

Recording Duration: How long to record when a vehicle is detected (recommended: 5-10 seconds)

üîç ALPR Settings

Detection Model: Choose based on your device:

  • YOLO v9 Tiny 640 - Best for most users (balanced speed/accuracy)
  • YOLO v9 Small 608 - Best accuracy (requires more power)

OCR Model: Reads the plate characters

  • CCT Small - Recommended for most users

Minimum Confidence: Set between 0.8-0.95. Higher = fewer false positives

üéõÔ∏è Camera Control (ISP Settings)

Fine-tune your camera's image quality for better plate recognition.

Exposure Mode
  • Auto: Let the camera decide (good for general monitoring)
  • Manual: Lock exposure settings for fast-moving vehicles
Day/Night Mode
  • Auto: Switches automatically based on light
  • Color: Always use color (not recommended for ANPR)
  • Black & White: Best for plate recognition - shutter/gain controls work properly
‚ö†Ô∏è Important: Use Black & White mode for best results! Many Reolink cameras don't properly apply manual shutter/gain settings in Color mode, resulting in blurry plates. Black & White mode ensures your manual exposure settings are respected.
Shutter Speed (Manual Mode Only)

Lower values = faster shutter = sharper plates on moving vehicles
Tip: For moving cars, try Min=1, Max=4

Gain (Manual Mode Only)

Controls brightness. Higher = brighter but noisier
Tip: Keep Max at 40-60 for cleaner images

‚öôÔ∏è Advanced: Before/After Recording

Before Recording: Temporarily change camera settings when a vehicle is detected (e.g., fast shutter for clear plates)

After Recording: Restore normal settings when done

Leave disabled if you're unsure - the default settings work well for most setups

üè† Home Assistant

This is where it gets powerful! You can trigger ANY Home Assistant automation:

  • Open garage door when your car arrives
  • Turn on lights when a vehicle is detected
  • Send alerts for unknown plates
  • Log visitors to a spreadsheet
  • Play announcements through smart speakers

Setup:

  1. In Home Assistant: Settings ‚Üí Automations ‚Üí Create Automation
  2. Choose "Webhook" trigger
  3. Copy the webhook URL
  4. Paste in ReolinkANPR config page
  5. Add conditions and actions (e.g., "if plate_number == 'ABC123' then open garage")

Example webhook data:

{
  "plate_number": "ABC123",
  "confidence": 0.975,
  "image_path": "images/..."
}
üí° Quick Tips
  • Start with default settings and adjust only if needed
  • Always use Black & White mode - shutter/gain work properly (Color mode often ignores manual settings)
  • For fast-moving vehicles: Manual exposure + fast shutter (1-4) + B&W mode
  • Test different settings and check the dashboard for results
  • Lower confidence = more detections (but more false positives)
  • Automate everything with Home Assistant! Open garage, turn on lights, etc.
